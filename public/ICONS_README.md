# PWA Icons

## Required Icons

This directory should contain the following PWA icons:

- `icon-192x192.png` - 192x192 pixels (required)
- `icon-512x512.png` - 512x512 pixels (required)

## Generating Icons

You can generate these icons from a source image using:

1. **Online Tools:**
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. **Command Line:**
   ```bash
   # Using ImageMagick
   convert source.png -resize 192x192 icon-192x192.png
   convert source.png -resize 512x512 icon-512x512.png
   ```

3. **Design Tools:**
   - Figma, Adobe Illustrator, etc.

## Icon Guidelines

- **Format:** PNG with transparency
- **Style:** Square with rounded corners (optional)
- **Colors:** Match brand colors
- **Content:** Company logo or app icon
