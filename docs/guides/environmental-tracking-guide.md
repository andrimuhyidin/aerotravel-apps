## 1. Logging Waste After Trip

### When to Log
- **After every trip completion**
- **Before marking trip as complete**
- System will remind you if not logged

### How to Log Waste

1. **Open Trip Details** in Guide App
2. **Scroll to "Environmental Data" section**
3. **Tap "Log Waste"** button
4. **Select Waste Type** and enter quantity:

| Waste Type | Unit | Examples |
|------------|------|----------|
| Plastic | kg or pieces | Bottles, bags, packaging |
| Organic | kg | Food waste, fruit peels |
| Glass | kg or pieces | Bottles |
| Hazmat | kg | Batteries, fuel containers |

5. **Select Disposal Method**:
   - âœ… Recycling (preferred)
   - Landfill
   - Incineration
   - Ocean (accidental loss only)

6. **Take Photo** (optional but recommended)
7. **Add Notes** if needed
8. **Submit**

### Tips for Accurate Logging
- Use boat scale if available
- Estimate if exact weight unknown
- Count items if weighing not possible
- Take photos for audit purposes

---

## 2. Fuel Consumption (Admin Only)

Fuel data is logged by Operations/Admin staff, not guides.

**Admin Process**:
1. Review engine log after trip
2. Note liters consumed
3. Enter in admin dashboard
4. System auto-calculates CO2 emissions

**Emission Factors** (IPCC 2006):
- Diesel: 2.68 kg CO2 per liter
- Gasoline: 2.31 kg CO2 per liter

## 6. Frequently Asked Questions

**Q: What if I forget to log waste?**  
A: Log it within 48 hours. Contact operations if longer.

**Q: What if waste was minimal/zero?**  
A: Still log it! Enter "0" to show trip was completed cleanly.

**Q: Can I edit logged waste?**  
A: No. Contact operations manager if correction needed.

**Q: What about water usage?**  
A: Not required currently. May be added in future.

**Q: Is this mandatory?**  
A: Yes, part of trip completion process and ISO 14001 compliance.
