# OHS-004: Incident Reporting

**Document ID:** OHS-004  
**Version:** 1.0  
**ISO Standard:** ISO 45001:2018 (Clause 10.2)

## Purpose
Document and investigate all incidents to prevent recurrence.

## Reportable Events
1. **Fatal/Serious Injury**: Immediate report
2. **Minor Injury**: Within 24 hours
3. **Near-Miss**: Within 48 hours
4. **Property Damage**: Within 48 hours
5. **Environmental Incident**: Within 24 hours

## Reporting Process

### Step 1: Immediate Actions
- Ensure safety
- Provide first aid/assistance
- Secure scene if investigation needed
- Take photos if safe to do so

### Step 2: Report via Guide App
- Open Incident Report form
- Select incident type
- Describe chronology
- Add photos/voice notes
- Submit digital signature
- Submit report

### Step 3: Notification
System auto-notifies:
- Operations manager
- OHS manager
- Insurance (for injuries)
- Senior management (for serious incidents)

### Step 4: Investigation
Conducted by OHS manager for:
- All serious injuries
- Near-misses with high potential
- Repeat incidents

Investigation includes:
- Witness interviews
- Scene examination
- Root cause analysis (5 Whys)
- Timeline reconstruction

### Step 5: Corrective Actions
- Immediate corrections
- Root cause mitigation
- Process improvements
- Training updates
- Equipment modifications

### Step 6: Follow-up
- Verify corrective actions implemented
- Check effectiveness (30-90 days)
- Update risk assessments
- Share lessons learned

## Reporting Metrics
- Time to report: Target < 24 hours
- Investigation completion: < 5 days
- Corrective action closure: < 30 days
- Repeat incidents: Target 0%

## No-Blame Culture
Reporting is encouraged; focus is on learning and prevention, not punishment (except for willful misconduct).
