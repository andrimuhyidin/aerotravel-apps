# OHS-003: Emergency Response

**Document ID:** OHS-003  
**Version:** 1.0  
**ISO Standard:** ISO 45001:2018 (Clause 8.2)

## Purpose
Ensure rapid and effective response to emergencies.

## Emergency Types
1. Medical (injury, illness)
2. Marine (man overboard, equipment failure)
3. Weather (storm, typhoon)
4. Security (theft, assault)
5. Environmental (fuel spill)

## Response Procedures

### Medical Emergency
1. Assess situation safely
2. Provide first aid
3. Activate SOS via app
4. Contact emergency services
5. Evacuate if needed
6. Document incident

### Man Overboard
1. Shout "Man Overboard!"
2. Throw flotation device
3. Keep visual contact
4. Circle back slowly
5. Recover person
6. Check medical condition

### Severe Weather
1. Monitor weather updates
2. Seek safe harbor if possible
3. Secure equipment
4. Brief passengers
5. Contact operations center
6. Follow captain's instructions

## Communication
- **SOS App**: Primary alert system
- **Radio**: VHF Channel 16 (emergency)
- **Phone**: Operations hotline (24/7)
- **Location**: GPS coordinates from app

## Emergency Equipment
- Life jackets (1 per person + spares)
- First aid kit (checked before each trip)
- Fire extinguisher
- Communication devices
- Emergency flares
- Life raft (capacity-appropriate)

## Training
- Emergency drills: Quarterly
- First Aid certification: Every 2 years
- Emergency procedure review: Annual
